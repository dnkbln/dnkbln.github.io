Podcast Settings Tab Description

Zusammenfassung

Hier werden die grundlegenden Daten zur Beschreibung und Identifizierung des Podcasts festgelegt.

Die Eingabe eines Titels, Untertitels, der Zusammenfassung und eines Titelbildes sind zwingend erforderlich.

Abbildung 9. Bildschirmfoto von den Podcast Settings Tab Description



Die gestaffelte Eingabe der Informationen zum Titel, Untertitel und Zusammenfassung gibt es auch bei den Episoden. Diese abgestufte Information soll es Nutzer des Feeds (z.B. Podcast-Clients, Playern, usw.) helfen die Informationen auf eine geeignete Art darzustellen. Ist eine Information bereits ausreichend mit Titel und Untertitel beschrieben, so sollte die Information nicht einfach in die ZUsammenfassung kopiert werden.

Tabelle 2. Description Elemente

Element

Aufgabe

Abbildung 10. Bildschirmfoto der Hilfe zum Tab Description

Am Anfang findest du eine kurze Hilfsanleitung für die Felder Title, Subtitle und Summary

Abbildung 11. Bildschirmfoto Eingabe Title



Der hier eingegebene Titel ist die Bezeichnung die zur Darstellung des Podcast in Podcast-Verzeichnissen (z.B. iTunes), Listen und anderen Anwendungen benutzt wird.

Abbildung 12. Bildschirmfoto Eingabe Subtitle



Mit dem Untertitel kann einer erster Ausblick darauf gegeben werden mit was sich der Podcast inhaltlich beschäftigt und was er vermitteln will.

Abbildung 13. Bildschirmfoto Eingabe Summary



Hier kann eine kurze Zusammenfassung über den Podcast angegeben werden. Diese kann aus einem oder mehreren Sätzen bzw. Absätzen bestehen.

Abbildung 14. Bildschirmfoto Eingabe Image



Hier kann ein Cover Art des Podcasts angegeben werden. Die Bilddatei wird mit Hilfe der Medienverwaltung von Wordpress hochgeladen und ausgewählt. Wird später in den Episoden kein eigenes Episodenbild angegeben, so wird das Cover Art des Podcasts auch für die Episoden verwendet. Die Bilddatei muss eine quadratische Auflösung haben und eine Mindestauflösung von 1400x1400 px. iTunes schlägt als Auflösung 3000x3000 px vor.

Abbildung 15. Bildschirmfoto Eingabe Mnemonic



Abbildung 16. Bildschirmfoto Eingabe Language



Hier kannst Du angeben in welcher Sprache der Podcast erstellt wird.

Abbildung 17. Bildschirmfoto Eingabe Type



Hier kannst Du in welcher Reihenfolge die Episoden dargestellt werden sollen. Es gibt zwei Optionen. In der Episodic Darstellung werden die letzten Folgen als erstes dargestellt. In der Serial Darstellung werden die Folgen in der Reihenfolge ihrer Erstellung präsentiert. Dies kann hilfreich für Geschichten, mehrere Staffeln oder Erzählungen sein.
