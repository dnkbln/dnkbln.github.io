Dashboard

Zusammenfassung

Das Dashboard zeigt Informationen zu den verwalteten Podcasts an als auch Informationen zum Podlove Projekt.

Abbildung 2. Bildschirmfoto vom Dashboard



Das Dashboard enthält 4 Boxen. At a glance und Validate Podcast Files enthalten Informationen über die publizierten Podcasts und die Boxen About und Podlove News stellen Informationen über das Podlove Projekt bereit.

Abbildung 3. Bildschirmfoto der Screen Optionen im Dashboard



Erinnerung: Mit Hilfe der Screen Optionen kann die Sichtbarkeit der einzelnen Boxen ein- und ausgeschaltet werden.

Tabelle 1. Dashboard Elemente

Box

Aufgabe

Abbildung 4. Bildschirmfoto der At a glance Box im Dashboard



Hier findest du eine Zusammenfassung der Episoden. Außerdem wird angegeben wie viele Episoden bereits veröffentlicht wurden, privat sind, noch veröffentlicht werden können und als Entwurf vorhanden sind. Des weiteren gibt es eine kleine Statistik mit der durchschnittlichen Länge einer Episode, der Gesamtlänge aller Episoden, der durchschnittlichen Media-Filegröße, der Gesamtgröße der Media-Files und dem durchschnittliche Abstand in dem neue Episoden veröffentlicht werden.

Abbildung 6. Bildschirmfoto der About Box im Dashboard

Abbildung 7. Bildschirmfoto der Podlove News Box im Dashboard



Hier findest du Hinweise zur Veröffentlichung neuer Versionen.
