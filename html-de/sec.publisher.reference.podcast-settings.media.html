Podcast Settings Tab Media

Zusammenfassung

Hier wird der Ort der Mediendaten festgelegt.

Abbildung 4.11. Bildschirmfoto von den Podcast Settings Tab Media


Im Media-Tab wird der Speicherort der Mediendaten bestimmt. Diese Information nutzt der Publisher um die URL aller Media-Dateien zu bestimmen. Wird der Speicherort geändert (z.B. durch Provideränderung) so muss die Änderung nur hier vorgenommen werden. Gibt es am Speicherort bereits eine Datei, so kann im Browser wie folgt kontrolliert werden, ob der richtige Speicherort angegeben wurde. Dazu wird die URL aus dem Eingabefeld hier in die Adresszeile des Browsers kopiert und ein Filename von einem bereits existierenden Datei hinzugefügt. Nach der Bestätigung der Eingabe sollte der Browser diese Datei anzeigen.

Ohne eine Änderung der Wordpress Standardeinstellungen kann der Mediaordner von Wordpress nicht für die Mediendaten für einen Podcast verwendet werden. Wird die folgende Einstellung deaktiviert so kann der Wordpress Mediaordner auf für Podcast genutzt werden. Dies wäre die einfachste Möglichkeit um einen Podcast zu starten.

Hier wird der Ort angegeben an dem die Media-Files gespeichert werden. Dabei handelt es sich um die Basis-URL. Alle Files sollen in diesem Verzeichnis gespeichert werden. Die einzelnen Episoden sollen nicht in eigene Verzeichnisse gespeichert werden.
