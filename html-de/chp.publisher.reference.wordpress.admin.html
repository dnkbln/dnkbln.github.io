Kapitel 1. Wordpress Admin

Zusammenfassung

Menüeinträge im Wordpress Admin Menu

Abbildung 1.1. Wordpress Admin Episodes und Podlove


Nach der Installation und Aktivierung des Podlove Publisher Plugins gibt es im Wordpress Admin Bereich zwei neue Menüeinträge Episodes und Podlove. Im Bereich Epidodes können neue Podcastfolgen erzeugt werden. Im Bereich Podlove erfolgen die globalen Einstellung zum Publizieren des Podcasts.
