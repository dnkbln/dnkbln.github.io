Podcast Settings Tab License

Zusammenfassung

Hier kann optional eine Lizenz für den Podcast ausgewählt werden.

Abbildung 5.23. Bildschirmfoto von den Podcast Settings Tab License


Im Lizenz-Tab kann optional eine Lizenz ausgewählt werden unter der der Podcast veröffentlicht wird.

Tabelle 5.7. Podcast License

Element

Aufgabe

Abbildung 5.24. Podcast Settings Lizenz Name

Hier kann der Name der gewünschten Lizenz angegeben werden.

Abbildung 5.25. Podcast Settings Lizenz URL

Hier kann eine URL angegeben werden. Dort sollte sich eine Beschreibung der angegebenen Lizenz finden.

Abbildung 5.26. Podcast Settings Lizenz Auswahl

Nach dem Ausklappen der Lizenzauswahl kann eine von 4 vordefinierten Lizenzen ausgewählt werden. Wird eine Lizenz ausgewählt, so wird automatisch der Name und die zugehörige URL ausgefüllt. Wurden hier Daten hinterlegt, so können diese Daten später in den Template verwendet werden.

Abbildung 5.27. Podcast Settings Lizenz Preview

Wurde eine der aufgelisteten Lizenzen ausgewählt, so gibt es nach der Auswahl eine Vorschau auf eine mögliche Darstellung der Lizenz. Die Darstellung muss aber über die jeweiligen Template-Tags aktiviert werden.
