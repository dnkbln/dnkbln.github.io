Hier wird festgelegt welcher Player gegebenfalls benutzt wird.

Tabelle 5.3. Player Einstellungen

Element

Aufgabe

Abbildung 5.15. Podcast Settings Player Selection

Wird später bei der Erstellung der Episoden ein Webplayer eingebunden, so wird der Webplayer verwendet, der hier ausgewählt wurde. Es stehen drei Webplayer zur Auswahl der Podlove Web Player 4 , der Podlove Web Player 5 und der Podigee Podcast Player.

Achtung

Der Podlove Web Player 4 ist veraltet. Es kann sein, dass in einer der zukünftigen Versionen dieser entfernt wird.

Achtung

Soll der Podlove Web Player 5 benutzt werden, so ist dieser als eigenständiges Plugin in Wordpress zu installieren.


Für jeden der gewählten Player gibt es verschiedene Möglichkeiten der Anpassunge. In den folgenden Tabellen sind die jeweiligen Einstellungen aufgelistet.

Tabelle 5.4. Einstellungen für den Podlove Web Player 4

Element

Aufgabe

Abbildung 5.16. Hauptfarbe

Im Podlove Web Player 4 wird das Farbschema von einer Haupt- und einer Nebenfarbe abgeleitet. Hier wird die Hauptfarbe festgelegt. Die Angabe der Farbe kann in Hex, RGB oder RGBA erfolgen. z.B. Hex: f5efe0, RGB: 245,239,224, RBGA: 245,239,224,0.9

Abbildung 5.17. Nebenfarbe

Im Podlove Web Player 4 wird das Farbschema von einer Haupt- und einer Nebenfarbe abgeleitet. Hier wird die Nebenfarbe festgelegt. Die Angabe der Farbe kann in Hex, RGB oder RGBA erfolgen. z.B. Hex: f5efe0, RGB: 245,239,224, RBGA: 245,239,224,0.9

Abbildung 5.18. Sichtbarkeit der Web Player 4 Componenten

Hier wird festgelegt welche Teile vom Podlove Web Player 4 angezeigt werden.

Abbildung 5.19. Spracheinstellung

Ist diese Option aktiv, wird die Spracheinstellung des Podcasts verwendet. Ist die Option inaktiv, wird die Spracheinstellung des Browsers verwendet.
