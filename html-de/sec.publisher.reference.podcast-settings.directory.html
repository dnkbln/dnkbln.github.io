Podcast Settings Tab License

Zusammenfassung

Hier können Angaben für Podcast-Directories z.B. iTunes gemacht werden.

Abbildung 4.28. Bildschirmfoto von den Podcast Settings Tab Directory


Im Directory Tab werden Informationen für die Podcast Directories angegeben. Die Daten werden im Feed teilweise in speziellen Einträgen verwendet. In der folgenden Tabelle sind die Einträge mitangegeben,

Tabelle 4.8. Podcast Directory

Element

Aufgabe

Abbildung 4.29. Podcast Settings Directory Author

Hier wird der Author des Podcasts angegeben. Diese Information ist öffentlich und findet sich zum Beispiel im iTunes Verzeichnis. An dieser Stelle könnte auch ein Künstlername verwendet werden. Diese Angabe füllt im Feed den Eintrag itunes:author

Abbildung 4.30. Podcast Settings Directory Publisher

Angabe des Podcast Publishers

Abbildung 4.31. Podcast Settings Directory Publisher URL

Angabe einer URL mit Informationen zum Publisher des Podcasts.

Abbildung 4.32. Podcast Settings Directory Besitzer

Angabe wer für diesen Podcast zu ständig. Diese Angabe wird verwendet wenn Podcastverzeichnisse mit dem Podcastbeteiber in Kontakt treten wollen.

Abbildung 4.33. Podcast Settings Directory Besitzer E-Mail Adresse

Angabe der E-Mail Adresse des Podcastverantwortlichen. Die eMail Adresse verwenden die Podcastverzeichnisse um den Verantwortlichen des Podcasts zu kontaktieren.

Abbildung 4.34. Podcast Settings iTunes Categorie

Im iTunes Podcastverzeichnis werden die Podcasts mit Hilfe von Kategorien in Bereiche aufgeteilt. Hier kann eine möglichst gut passende Kategorie gewählt werden.

Abbildung 4.35. Podcast Settings iTunes Explicit

Werden im Podcast bestimmte Themen (z.B. Gewalt, Sexualität) angesprochen so kann dies hier vermerkt werden.

Abbildung 4.36. Podcast Settings Directory Complete

Ist der Podcast abgeschlossen und sind keine weiteren Folgen geplant, so kann dies hier dem Podcastverzeichnissen mitgeteilt werden.
