Podcast Settings Tab Affiliate

Zusammenfassung

Hier können für Amazon unfd Thomann Affiliate Ids hinterlegt werden.

Abbildung 4.37. Bildschirmfoto von den Podcast Settings Tab Affiliate


Im Affiliate Tab können die Affiliate Ids des Podcast-Betreibers hinterlegt werden. Diese Ids werden nur genutzt wenn das Shownotes Modul aktiv ist. Werden in den Shownotes Links zu Amazon oder Thomann gesetzt und wurden hier Affiliate Ids hinterlegt, so werden die Links so angepasst, dass die Affiliate Ids integriert werden.
